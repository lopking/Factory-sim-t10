⚡ What is a "mines ratio"? Each machine needs a steady supply of ore/fluid to run non-stop. The ratio tells you how many mines (or extractors) per machine you need so it never stops. A ratio of 2.67 mines means you need 3 mines (round up) to guarantee one machine stays busy 100% of the time.
