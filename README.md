Factory Sim — Tier 10 Full Chain Viewer & Calculator
A comprehensive suite of tools designed for Factory Simulator Tier 10 optimization. This project helps players visualize complex production trees and calculate exact machine requirements for high-level automation.

🚀 Overview
Managing Tier 10 production involves deep dependency trees and precise resource balancing. This tool provides a Full Chain Viewer to map out every step from raw extraction to the final product, alongside dedicated Ratio and Calculator modules to ensure maximum efficiency.

✨ Features
Interactive Chain Viewer: Select any T10 item to generate a complete visual ingredient tree.

Visual Status Coding: * 🟢 Raw Resources: Mines, farms, and extractors.

🔵 Crafted Items: Machine-processed components.

🟠 Byproducts: Secondary outputs to manage.

Chain Summary Panel: Automatically tallies the total number of unique machines and raw resources required for the selected item.

Collapsible Nodes: Manage complex trees by expanding or collapsing specific branches.

Integrated Math Tools: Quick access to ratio sheets and production calculators directly from the main interface.

🛠️ Tech Stack
HTML5: Structure for the interactive tree and UI.

CSS3: Custom styling for the node-based visualization and responsive panels.

JavaScript (ES6): Logic for tree generation, search filtering, and calculation algorithms.

How to Use
1. Select an Item: Use the dropdown menu or search bar to find a Tier 10 item.

2. Analyze the Tree: View the dependencies and click [−] to collapse sections you have already automated.

3. Check the Summary: Review the "Chain Summary" on the right to see exactly how many machines you need to place.

4. Balance Ratios: Use the navigation links at the top to switch to the Calculator or Ratios page for fine-tuning.
